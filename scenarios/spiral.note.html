Spiral maze across all players — a single continuous spiral

One Archimedean spiral wall winds from the outer edge all the way to the Nexus, crossing every player's section. It leaves a single smooth spiral corridor, so a unit spirals continuously inward — getting steadily closer to the center the whole way (no concentric rings, no pockets).

The wall is defined by a field f(r,θ) = (R0−r)/pitch + dir·θ/2π whose integer contours are the spiral windings; we wall every grid edge where floor(f) changes — cardinal and diagonal, so the wall is solid — and apply the same test across the seams so the wall joins across the borders between players' rotated grids. See the spiral-trace tab for a single unit tracked and plotted (radius vs time) to confirm the motion is one smooth, continuous spiral.
